Adds a custom signature to your RSS feed items.

== Description ==

This plugin allows you to automatically add a custom signature to the end of each item in your WordPress RSS feed. The signature includes a link back to your website and mentions your blog name, helping to attribute content correctly when syndicated.

== Installation ==

1. Upload the `add-signature-to-rss` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The signature will automatically be added to your RSS feed items. There are no settings to configure.
